## 🚀 クイックスタート

### 前提条件

始める前に、以下がインストールされていることを確認してください：
- Node.js (v18以上)
- npm または yarn
- Git

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/yourusername/yourproject.git

# プロジェクトディレクトリに移動
cd yourproject

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

`http://localhost:3000` にアクセスすると、アプリケーションが動作しています！🎉
